# 🎓 Grade Calculator

A simple and efficient Grade Calculator built using Python.  
This project helps students calculate their total marks, percentage, and grade based on the marks entered.

---

## 🚀 Features

- 📊 Calculates total marks
- 📈 Computes percentage automatically
- 🏆 Assigns grade based on percentage
- ⚡ Simple and easy-to-use interface
- 🧠 Beginner-friendly project for learning Python

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook

- Execute all cells

---

## 🧮 How It Works

1. User enters marks for subjects  
2. Program calculates:
   - Total marks  
   - Percentage  
3. Based on percentage, grade is assigned  

---

## 🎯 Example

Input:

Maths: 90

Science: 85

English: 88

Output:

Total: 263

Percentage: 87.6%

Grade: A
